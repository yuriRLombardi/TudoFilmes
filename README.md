# 🎬 TudoFilmes

Projeto acadêmico desenvolvido como Trabalho de Conclusão de Curso (TCC), com o objetivo de planejar e estruturar um sistema web para gerenciamento de catálogo de filmes.

---

## 🎯 Objetivo

O projeto foi concebido para simular uma plataforma de organização e visualização de filmes, permitindo cadastro, consulta e gerenciamento de informações de forma centralizada.

---

## 💡 Proposta da Solução

O **TudoFilmes** propõe um sistema capaz de:

- Gerenciar um catálogo de filmes
- Permitir cadastro, edição e remoção de registros pelo administrador
- Exibir informações detalhadas dos filmes
- Organizar dados de forma estruturada

---

## ⚙️ Funcionalidades Planejadas

- Cadastro de filmes  
- Listagem de filmes  
- Edição e remoção (CRUD)  
- Visualização de detalhes  

---

## 🛠️ Tecnologias Utilizadas

- PHP  
- HTML  
- CSS  
- JavaScript  
- MySQL  

---

## 🧠 Arquitetura

O sistema foi estruturado seguindo organização em camadas, separando:

- Lógica de aplicação (PHP)  
- Interface (HTML/CSS/JS)  
- Persistência de dados (MySQL)
